# Employee Management & Leave Tracking Module

### Project Overview:
This module is built using the Frappe Framework to manage Employee records and track Leave Requests with automated date validation.

### Features Implemented:
- **Employee Management**: CRUD operations for employee data.
- **Leave Requests**: Logic to ensure 'To Date' is not before 'From Date'.
- **Reporting**: Integrated reporting with CSV export capability.

### Files Included:
- `Doc Employee.json`: Schema for Employee DocType.
- `Leave_Request.json`: Schema for Leave Request DocType.
- `leave_request.py`: Validation logic for leave dates.
